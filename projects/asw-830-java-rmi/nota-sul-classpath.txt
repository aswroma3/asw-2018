il classpath va "resettato" se: 
- il registry è avviato dalla radice del file java 
- si usa il codebase 
- il registry è avviato dal programma java per il server 

altrimenti va settato 

inoltre, il classpath usa come separatore: 
- : su linux 
- ; su windows
